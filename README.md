This is an Url Shortener web app made using :
  - Frontend
      - Reactjs
      - Typescript
      - Tailwind CSS
      - Vite
  - Backend
      - Nodejs
      - Express
      - MongoDB
      - Npm

To run frontend-

npm install ---> npm run dev
  

To run backend:

- Setup mongoDb database in .env file
- npm install ---> npm start
